📱 在线构建 APK

无需安装 Android SDK，直接在线打包

🚀 快速开始：使用 GitHub Pages 托管

  1. 创建 GitHub 账号并登录 github.com
  2. 创建新仓库，上传整个项目文件夹
  3. Settings → Pages → Source: Deploy from branch → main
  4. 等待部署完成，获得网址如: https://你的用户名.github.io/仓库名
  5. 使用下方在线服务打包该网址

1. PWABuilder 推荐

微软官方 PWA 打包服务，支持 Android、iOS、Windows 多平台。

✅ 免费 | ✅ 微软官方 | ✅ 多平台支持

⚠️ 需要先部署网页

打开 PWABuilder →

2. AppGyser

免费将网站转换为安卓 App，简单易用。

✅ 完全免费 | ✅ 无需技术基础

⚠️ 有广告 | ⚠️ 需要注册

打开 AppGyser →

3. WebIntoApp

一键将网页转换为安卓 App，支持自定义图标。

✅ 免费版可用 | ✅ 无需注册

⚠️ 免费版有水印

打开 WebIntoApp →

4. Andromo

专业的 App 制作平台，支持更多自定义选项。

✅ 功能丰富 | ✅ 专业级

⚠️ 付费服务

打开 Andromo →

💡 最简单的方法

如果只是自己使用，直接用手机浏览器打开网页，点击"添加到主屏幕"即可获得 App 体验！
