📱 在线打包成安卓 APK

方法一：PWABuilder（推荐）

微软官方提供的 PWA 打包服务，可以将网页直接打包成安卓 APK。

步骤：

1. 先将本应用部署到网上（GitHub Pages、Vercel、Netlify 等）

2. 访问 PWABuilder，输入你的网址

3. 点击生成 APK 并下载

打开 PWABuilder →
提示：如果没有服务器，可以先用 GitHub Pages 免费托管：
  1. 创建 GitHub 仓库
  2. 上传所有文件
  3. Settings → Pages → 选择 main 分支
  4. 获得免费网址：https://你的用户名.github.io/仓库名

方法二：GoNative

快速将网站打包成原生 App，支持 iOS 和 Android。

打开 GoNative →

方法三：WebIntoApp

完全免费的网页转 App 服务，无需注册。

打开 WebIntoApp →

方法四：直接添加到主屏幕（PWA）

如果不想打包，可以直接将网页添加到手机主屏幕，效果和 App 一样：

安卓 Chrome：

  1. 用 Chrome 打开网页
  2. 点击右上角菜单（三个点）
  3. 选择"添加到主屏幕"

iPhone Safari：

  1. 用 Safari 打开网页
  2. 点击底部分享按钮
  3. 选择"添加到主屏幕"

方法五：本地打包（需要管理员权限）

如果 npm 有权限问题，需要以管理员身份运行 PowerShell：

  1. 右键点击 PowerShell，选择"以管理员身份运行"
  2. 运行命令: npm cache clean --force
  3. 运行命令: npm install -g cordova
  4. 参考 ANDROID_PACKAGING_GUIDE.md 完成打包
